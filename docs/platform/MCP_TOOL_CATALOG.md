- **Tool class**: `Ai::Tools::KnowledgeGraphTool`
- **Permission**: ai.agents.read

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `node_id` | string | Yes | Node ID |

### `list_graph_nodes`

List knowledge graph nodes with optional filters and pagination

- **Tool class**: `Ai::Tools::KnowledgeGraphTool`
- **Permission**: ai.agents.read

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `node_type` | string | No | Filter by node type |
| `entity_type` | string | No | Filter by entity type |
| `query` | string | No | Search query |
| `knowledge_base_id` | string | No | Filter by knowledge base |
| `page` | integer | No | Page number |
| `per_page` | integer | No | Results per page (default 20) |

### `get_graph_neighbors`

Get neighboring nodes and edges for a specific graph node

- **Tool class**: `Ai::Tools::KnowledgeGraphTool`
- **Permission**: ai.agents.read

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `node_id` | string | Yes | Node ID |
| `depth` | integer | No | Traversal depth (default 1, max 5) |
| `relation_types` | array | No | Filter by relation types |

### `graph_statistics`

Get knowledge graph statistics (node counts, edge counts, etc.)

- **Tool class**: `Ai::Tools::KnowledgeGraphTool`
- **Permission**: ai.agents.read
- **Parameters**: none

### `get_subgraph`

Extract a subgraph containing the specified nodes and their connections

- **Tool class**: `Ai::Tools::KnowledgeGraphTool`
- **Permission**: ai.agents.read

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `node_ids` | array | Yes | Array of node IDs |

### `extract_to_knowledge_graph`

Extract entities and relationships from text and add them to the knowledge graph

- **Tool class**: `Ai::Tools::KnowledgeGraphTool`
- **Permission**: ai.agents.read

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | Text to extract entities and relations from |
| `source_label` | string | No | Label for the extraction source |
